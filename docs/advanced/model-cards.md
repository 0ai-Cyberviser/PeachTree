# Model Cards

Generate comprehensive model card documentation for datasets.

## Creating Model Cards

```bash
peachtree model-card dataset.jsonl \
  --name "MyDataset" \
  --version "1.0.0" \
  --output model-card.md
```

## Card Sections

- Dataset overview
- Data collection methodology
- Intended use
- Training recommendations
- Limitations and biases
- License and attribution
- Ethical considerations
- Risk analysis
