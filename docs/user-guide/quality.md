# Quality Scoring

Assess and filter dataset records by quality metrics.

## Overview

Quality scores (0.0 - 1.0) measure record fitness for training.

```bash
peachtree quality dataset.jsonl --min-score 0.8
```

## Metrics

- Content length and diversity
- Documentation quality
- Code structure and style
- License clarity
- Metadata completeness
