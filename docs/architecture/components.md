# Components

Detailed guide to PeachTree core components.

## DatasetBuilder
Main engine for converting source documents to training records.

## SafetyGate
Multi-layer filtering for secrets, licenses, and unsafe content.

## PolicyPackEvaluator
Composable policy evaluation system.

## RegistryBuilder
Artifact discovery and indexing.

## ReleaseBundle
Packaging for distribution with SBOM and signatures.

See [Architecture](design.md) for detailed information.
